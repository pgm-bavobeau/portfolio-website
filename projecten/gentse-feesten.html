Project: Gentse Feesten 2022

Voor het vak User Interface Design heb ik de opdracht gekregen om de Gentse Feesten website na te maken. Het doel van het project is om een responsieve website te creëren met behulp van HTML, CSS en JavaScript.

Het project

Voor mijn project voor het vak User Interface Design kregen we de opdracht de Gentse Feesten website na te maken. Het doel was om een volledig responsive website te ontwikkelen met behulp van HTML, CSS en JavaScript.

Ik begon met het analyseren van de oorspronkelijke website om de structuur ervan te begrijpen. Vervolgens heb ik de HTML-code geschreven om de pagina's en secties van de website op te bouwen. Ik creëerde een duidelijk hoofdmenu met links naar alle belangrijke pagina's, dat op elke pagina werd weergegeven.

Daarna ging ik aan de slag met het stijlen van de website met CSS. Ik bestudeerde de visuele elementen van de originele website, zoals kleuren, typografie en lay-out, en heb deze stijlkenmerken toegepast met behulp van CSS-regels. Ik maakte gebruik van responsive CSS-eigenschappen om ervoor te zorgen dat de website er goed uitzag op verschillende schermformaten.

Om de website interactief te maken, heb ik JavaScript gebruikt. Ik heb gekeken naar de functionaliteiten van de originele website, zoals de evenementenagenda, filters en zoekfunctionaliteit, en ik heb deze geïmplementeerd met JavaScript. Het was belangrijk voor mij om ervoor te zorgen dat deze functionaliteiten goed werkten op zowel desktop- als mobiele apparaten.

Tijdens het ontwikkelen van de website heb ik ook veel aandacht besteed aan de gebruikerservaring. Ik heb ervoor gezorgd dat de navigatie intuïtief was en dat gebruikers gemakkelijk de inhoud konden vinden en lezen. Ik heb de responsieve eigenschappen van de website uitvoerig getest op verschillende apparaten om er zeker van te zijn dat alle elementen correct werden weergegeven.

Tot slot heb ik me gehouden aan de principes van User Interface Design. Ik heb gezorgd voor een consistente en aantrekkelijke visuele stijl, duidelijke taalgebruik en gemakkelijke interactiemogelijkheden voor gebruikers. Daarnaast heb ik ervoor gezorgd dat de website toegankelijk was voor mensen met verschillende vaardigheden, bijvoorbeeld door te zorgen voor een goede contrastverhouding tussen tekst en achtergrond.

Ik ben trots op het resultaat van mijn project, waarin ik de Gentse Feesten website heb nagemaakt als een volledig responsive website met behulp van HTML, CSS en JavaScript. Het heeft me geholpen mijn vaardigheden op het gebied van User Interface Design verder te ontwikkelen en een gebruiksvriendelijke en aantrekkelijke ervaring te creëren voor de gebruikers.
